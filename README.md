# 🎨 Blog HTML Editor Pro Master

> **전문적인 블로그 포스팅을 위한 고성능 HTML 편집기**

이 프로젝트는 블로거들이 복잡한 HTML 코드를 직접 다루지 않고도, 세련된 디자인의 블로그 포스팅을 작성할 수 있도록 돕는 실시간 웹 에디터입니다. 특히 쿠팡 파트너스, 상세 페이지, 정보성 포스팅에 최적화된 도구들을 제공합니다.

## ✨ 주요 기능

### 1. 강력한 편집 & 레이아웃
- **실시간 분할 뷰**: 좌측은 코드 에디터(CodeMirror), 우측은 실시간 미리보기.
- **줌 컨트롤**: 에디터 폰트 크기 및 미리보기 배율 조절 가능.
- **반응형 테스트**: 데스크톱, 태블릿, 모바일 뷰포트 즉시 전환.
- **다크 모드**: 시력 보호를 위한 툴바/에디터/미리보기 다크 모드 지원.

### 2. 생산성 도구
- **20가지 프리셋 테마**: 클릭 한 번으로 H2, H3, 버튼 색상을 일괄 변경.
- **자동 목차(TOC) 생성**: 본문을 스캔하여 이동 링크가 포함된 목차 자동 삽입.
- **SEO JSON-LD 생성기**: 검색 엔진 최적화를 위한 Article/FAQ 구조화 데이터 생성.
- **콘텐츠 삽입 도구**: 제목, 이미지, 링크, 쿠팡 스타일 버튼, 공정위 문구 등 원클릭 삽입.

### 3. 정밀 제어
- **양방향 동기화**: 미리보기 요소를 클릭하면 코드 에디터의 해당 위치로 자동 스크롤 및 하이라이트.
- **자동 저장**: LocalStorage를 활용하여 브라우저를 닫아도 소스코드 및 설정값 보존.
- **실행 취소/다시 실행**: 에디터 내 강력한 Undo/Redo 지원.

## 🚀 시작하기

이 프로젝트는 서버가 필요 없는 **Single-file HTML**로 구성되어 있습니다.

1. 이 저장소를 `Clone` 하거나 소스코드를 다운로드합니다.
2. `index.html` 파일을 크롬 등 브라우저로 엽니다.
3. 바로 글쓰기를 시작하세요!
